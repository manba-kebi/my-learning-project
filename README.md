# 这是一篇关于C++的学习笔记

这是我学习C++相关内容的一些笔记，像比较基础的一些内容，比如**类、重载、容器、算法（比如二分、动态、贪心、快排）这些思想在这里将不做学习了**，如果类、容器这些概念不清楚可以去看看 [牛客网](https://www.nowcoder.com/exam/intelligent?questionJobId=10&subTabName=intelligent_page&tagId=21009) ，如果算法这些东西不太懂可以去看看[leetcode](https://leetcode.cn/)。我现在状态是处于学了一年算法 与 3个月的C++编程语言的细节。虽然人外有人、天外有天，我知道我实力肯定是不够看的，但我还是想分享我的心得，因为在找实习、找工作之前，我一直以为学算法、数据结构，类的这些概念就可以了（但实际上这都是面试的一些基本问题吧），没想到企业对很多东西都是有个要求的。于是在这里我将进一步深挖C++的一些内容，也会将我的看法写在这里，希望能帮到大家，能让大家少走点弯路。

## 说一些心里话吧

在大一转大二的那个暑假，我收到了第十六届大学生数学竞赛的邀请（是全校高数分数前200名才能拿到的名额），当时我很激动，因为证明自己的时候到了（在刚上大学的时候，我的高数老师我自认为讲的还行，但是总是有点没讲明白，于是我就经常下课问她没讲到的地方，大概是第4次吧，她在麦克风开着的情况下对我说：”你不是没听课？“我当时懵了，因为不知道为什么她会说这句话，从那之后，我再也不敢问她题了（那个时候AI都没咋出来呢，也只有chatgpt这一个AI））。那时半个学期都给高数了，成果也还行，省一等奖、国二等奖。

到了寒假，我的上铺问我要不要报计算机二级，由于当时处于想拿证书证明自己的思想中，所以我报了个一直感兴趣但没怎么学过的python，也从那时慢慢有了计算机忧虑，我就让deep seek帮我制定计划，结果发现不是计划时间安排太过不正常、就是给的代码根本用不了（其实当时有点电子嘉豪了，觉得代码AI就可以搞定了，以为自己可厉害了）

后来，我就刷leetcode，也问过一些未来规划吧（说实话现在**咱们的计算机环境真差**，你想问ai(比如某包)，AI说你已经很棒了（我还啥都没学呢，就已经很棒了？），要不然就是给你完不成的规划。你想从网上看课程，但你不会翻墙、不会英语、没有好用的翻译软件（免费的那种），那么你从网上看到的课程都是两三年前教过的了，已经没有商用价值了，要不然就是每节课都给你放一半，想学就来花钱上网课），欸，我不知道其他领域什么样，但计算机领域防人真有点像防贼一样了啊。

直到半个月前（2026.4.15）左右，我用了朋友的chatgpt帮我规划了一下（因为当时真挺迷茫的）,结果发现还有这么多高级思想我还没有学到，给的资料已经是超乎我的预期了，真的是很无奈啊。

所以现在，我写下了这个项目，希望能帮助到后来的人。我还问我的好朋友呢，你说会有人不仅看不上还会来吐槽我两句吗，他回答是：为众人抱薪者，不可使其冻毙于风雪。希望这个文档能帮助到你们。



今天是7月15日。我转回来看看这篇README是否有什么不妥，因为过两天我准备拍个抖音去宣传我的项目与学习经历了。兜兜转转发现如果能让我的读者读到我的心里话，也是蛮有意思的，这样读者应该会有种真心相待的感觉吧。

这3个月内，也参加了一些竞赛，蓝桥杯大学B组 C++软件赛 省一等奖（国赛没获奖，主观原因：devC++版本太低了，写个二维容器**vector<vector\<int>>**还得注意 >> 中间加个空格，写成：**vector<vector\<int> >** 。客观原因：天下英雄太多了，我只是其中一位渺小的算法爱好者），计算机博弈竞赛 中国象棋 省一等奖（这次经历让我成长许多（不是学习方面的））。
