# 说一些心里话吧

在大一转大二的那个暑假，我收到了第十六届大学生数学竞赛的邀请（是全校高数分数前200名才能拿到的名额），当时我很激动，因为证明自己的时候到了（在刚上大学的时候，我的高数老师我自认为讲的还行，但是总是有点没讲明白，于是我就经常下课问她没讲到的地方，大概是第4次吧，她在麦克风开着的情况下对我说：”你不是没听课？“我当时懵了，因为不知道为什么她会说这句话，从那之后，我再也不敢问她题了（那个时候AI都没咋出来呢，也只有chatgpt这一个AI））。那时半个学期都给高数了，成果也还行，省一等奖、国二等奖。

到了寒假，我的上铺问我要不要报计算机二级，由于当时处于想拿证书证明自己的思想中，所以我报了个一直感兴趣但没怎么学过的python，也从那时慢慢有了计算机忧虑，我就让deep seek帮我制定计划，结果发现不是计划时间安排太过不正常、就是给的代码根本用不了（其实当时有点电子嘉豪了，觉得代码AI就可以搞定了，以为自己可厉害了）

后来，我就刷leetcode，也问过一些未来规划吧（说实话现在**咱们的计算机环境真差**，你想问ai，AI说你已经很棒了（我还啥都没学呢，就已经很棒了？），要不然就是给你完不成的规划。你想从网上看课程，但你不会翻墙、不会英语、没有好用的翻译软件（免费的那种），那么你从网上看到的课程都是两三年前教过的了，已经没有商用价值了，要不然就是每节课都给你放一半，想学就来花钱上网课），欸，我不知道其他领域什么样，但计算机领域防人真有点像防贼一样了啊。

直到半个月前（2026.4.15）左右，我用了朋友的chatgpt帮我规划了一下（因为当时真挺迷茫的）,结果发现还有这么多高级思想我还没有学到，给的资料已经是超乎我的预期了，真的是很无奈啊。

所以现在，我写下了这个项目，希望能帮助到后来的人。我还问我的好朋友呢，你说会有人不仅看不上还会来吐槽我两句吗，他回答是：为众人抱薪者，不可使其冻毙于风雪。希望这个文档能帮助到你们。
