# git操作

1. git add .	                                         把新写/修改的所有文件，告诉Git
2. git commit -m "写一句备注"           给文件拍个“版本快照”，永久保存
3. git push                                             把本地保存的版本，上传到GitHub上
4. git status                                           查看现在文件有没有改动、有没有保存
5. git log                                                 查看所有的提交历史
6. git pull                                                把GitHub上的最新内容拉到本地
7. git init                                                 新建一个本地Git仓库



# typora语法

## markdown基础语法

|   功能   |                  语法写法                   |                 效果说明                 |
| :------: | :-----------------------------------------: | :--------------------------------------: |
|   标题   | # 一级标题` / `## 二级标题` / `### 三级标题 | 用 `#` 号数量控制标题层级，最多支持 6 级 |
|   加粗   |                  **文字**                   |            文字会变成加粗效果            |
|   斜体   |                   *文字*                    |              文字会变成斜体              |
| 无序列表 |             - 列表项 / * 列表项             |               生成圆点列表               |
| 有序列表 |                  1.列表项                   |       生成数字列表，序号会自动调整       |
|  代码块  |      （前后各三个反引号，中间写代码）       |     代码会高亮、带背景框，排版很整齐     |
|   链接   |            [显示文字](链接地址)             |          点击文字就能跳转到链接          |
|   图片   |       ![图片说明](图片地址/本地路径)        | 插入图片，本地图片也可以上传到仓库一起看 |
|   引用   |                 > 引用内容                  |            生成带竖线的引用块            |
